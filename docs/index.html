About MediFlow

MediFlow is a solution combining a clinic management system for digital prescriptions and patient records with an automated pill dispenser for accurate and efficient medication delivery.

Simplifies workflows for doctors, pharmacists, and nurses, offering secure access controls to protect sensitive medical information. Built on a scalable and secure architecture, it is tailored to meet the evolving needs of modern healthcare facilities.

Solution Architecture

Integrated Healthcare Solution

MediFlow streamlines healthcare by integrating clinics, pharmacies, and an automated pill dispenser. Doctors enter prescriptions through a frontend system, linking them to the patientâ€™s RFID tag. This data is securely stored in the backend. Pharmacists access the prescription via their frontend, while the backend communicates with AWS IoT Core to send instructions to the pill dispenser, which then automatically dispenses the correct medication.

System Design

Cloud Architecture

CloudFront for content delivery
VPC with public and private subnets
EC2 instances for application hosting
RDS for secure database management
IoT Core for dispenser connectivity
Lambda & DynamoDB for serverless operations
Application Load Balancer for traffic management

Hardware

PCB Design

Our custom PCB design forms the heart of the automated dispensing system, featuring precision motor control, RFID integration, and sensor management for reliable medication dispensing.

ESP32 Microcontroller

Dual-core processor with WiFi and Bluetooth connectivity

Motor Control

Stepper motor driver DC 43A H-bridge for accurate dispensing

IoT Integration

Real-time connectivity with cloud services

RFID Module

Secure patient identification system withUSB RFID reader

Laser Receiver & Transmitter

Used KY008 Laser Module for accurate pill counting during dispensing operations

DHT11 Sensor

Monitors temperature and humidity of pill storage for optimal conditions
